Сервис вопросов и ответов

#Установка
1. Скачиваете через консоль: 
$ git clone https://github.com/Bludova/laravel.git

2. Устанавливаете: 
composer install

3. Настройте файл:  .env

4. Запустите миграцию: 
php artisan migrate
